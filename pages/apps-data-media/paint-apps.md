# Paint apps

## Sketchbook ~ Autodesk ~ https://www.autodesk.com/products/sketchbook/overview

* Bitmap only
* Download or app
* Can set image size
* Saves directly to Drive
* Many cool brushes
* UI complex though
* Pressure support

* https://en.wikipedia.org/wiki/Microsoft_Fresh_Paint

* Not pressure sensitive with Yoga Book
